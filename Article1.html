Stage à Permabita

Au cours de cette première année, j'ai effectué mon stage en distanciel au sein de l'association Permabita du 21/06/2021 au 30/07/2021. Lors de ce stage de 6 semaines, on m'a chargé de concevoir le site de l'association. J'avais comme contrainte de créer le site avec le CMS joomla!. Permabita est une association nouvellement créée ayant pour activité la construction d'habitations avec des matériaux écologiques.

Au début, j'ai décidé de faire une maquette du site avec figma. Cette maquette a évolué en fonction des retours de mon directeur de stage jusqu'à l'obtention de la version définitive. Une fois la maquette terminée, j'ai commencé à coder le site web en html et css avec visual studio code, car c'est plus facile pour faire des tests plutôt que de modifier directement sur joomla!.

Lorsque j'ai fini de coder le site en html et css, j'ai importé mon code html et css sur joomla!. Certains codes n'étant pas acceptés par joomla! j'ai dû apporter des modifications. N'ayant jamais utilisé joomla! j'ai ainsi dû m'auto former. Une fois le site terminé, j'ai fait une documentation personnalisée pour que même quelqu'un qui ne connaît pas joomla! puisse apporter des modifications sur le site web.

Personnellement, je n'ai pas aimé travailler avec joomla!, car je trouve que l'on est assez vite limité. Certaines choses que l'on peut faire simplement en code deviennent beaucoup plus complexe à faire sur joomla!. Pour conclure, en plus d'avoir découvert joomla!, ce stage m'a permis de découvrir comment procéder avec un client. Mon maître de stage était l'équivalent d'un client demandant d'effectuer une tâche avec un certain nombre de contraintes. Ainsi, nous faisions une réunion une fois par semaine où je lui montrais l'avancement du site et prenais en compte ses commentaires.
